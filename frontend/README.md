# Breezy Admin Frontend

React frontend for the Breezy Admin Panel - HubSpot Integration POC

## Setup

Dependencies are already installed. If you need to reinstall:

```bash
npm install
```

## Running the Application

Make sure the backend server is running on port 3001 first, then:

```bash
npm run dev
```

The frontend will be available at: http://localhost:5173

## Features (Part A)

- Customer registration form that syncs to HubSpot
- Contacts table showing all HubSpot contacts
- Auto-refresh after creating new contacts
- Expandable contact rows (ready for Parts B & C)

## Tech Stack

- React 18
- Vite
- CSS for styling
